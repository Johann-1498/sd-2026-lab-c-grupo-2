LAB 03 – Chat Cliente-Servidor



Cómo ejecutar el servidor



1\. Ubicarse en la carpeta del laboratorio

cd LAB03/PROPUESTOS

2\. Compilar los archivos

javac \*.java

3\. Ejecutar el servidor

java -cp . Server

4\. Resultado

El servidor iniciará correctamente y mostrará:



Server waiting for Clients on port 1500.
